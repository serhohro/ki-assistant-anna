# 🤖 Анна PRO v6.2 — Self-Improving AI Assistant

**Анна PRO** — это продвинутый локальный ИИ-ассистент на базе Streamlit и Ollama, предназначенный для обучения, анализа данных и помощи в разработке. Версия 6.2 включает 9 специализированных режимов работы и систему автоматического улучшения ответов.

## ✨ Основные возможности

* **9 интеллектуальных режимов:**
    * 🇩🇪 **German Coach** — изучение немецкого (от A1 до C1).
    * 📊 **Аналитика** — BI, статистика и поиск инсайтов.
    * 💻 **Программирование** — написание и оптимизация кода.
    * 📈 **Data Science** — ML, NLP и визуализация.
    * 🔍 **Код-ревью** — проверка безопасности и производительности.
    * 📝 **Оптимизация резюме** — подготовка к поиску работы.
    * ... и другие (Обучение, Собеседование, Вопрос-ответ).
* **Голосовое сопровождение:** Озвучка ответов с помощью gTTS (поддержка UK, RU, EN, DE).
* **Локальный запуск:** Полная приватность данных благодаря интеграции с Ollama.
* **Адаптивный интерфейс:** Динамическая смена эмоций Анны, поддержка видео и фото аватаров.
* **Система прогресса:** Отслеживание статистики вопросов и уровня пользователя.

## 🚀 Быстрый старт

### Требования
1.  **Python 3.8+**
2.  **Ollama** (установленная и запущенная локально)
3.  Любая LLM модель в Ollama (например, `llama3`, `mistral` или `qwen`).
